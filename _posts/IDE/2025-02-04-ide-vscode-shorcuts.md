회사에서는 WebStorm을 쓰고 있었는데, 유료인 Jetbrains를 사용하지 못 하는 환경에 대비하여 VSCode에 익숙해지고 앞으로는 VSCode를 사용해보려고 합니다. 더 유용한 단축키가 있다면 추천해주시면 감사하겠습니다.👍

## 에디터 내 기능

|   단축키    |                   설명                    |
| :---------: | :---------------------------------------: |
|    `⌘P`     |          빠른 열기, 파일로 이동…          |
| `⇧⌘P`, `F1` |            Sh명령 팔레트 표시             |
|    `⇧⌘N`    |                   새 창                   |
|    `⌘W`     |              애다터/탭 닫기               |
|   `⌘K F`    |            프로젝트 폴더 닫기             |
|    `⌘,`     |                사용자 설정                |
|   `⌘K ⌘S`   |          키보드 단축키 바로가기           |
|    `⌘B`     |  사이드바 포커싱 상태에서 사이드바 닫기   |
|    `⇧⌘E`    | 탐색기 보기 및 파일 탐색기 간 포커싱 전환 |
|    `⇧⌘F`    |              전체 검색 보기               |
|    `⌃⇧G`    |             소스 컨트롤 보기              |
|    `⇧⌘D`    |                디버그 보기                |
|    `⇧⌘X`    |            확장 프로그램 보기             |
|    `⇧⌘H`    |             전체 검색 바꾸기              |
|    `⇧⌘U`    |             결과창 패널 열기              |
|    `⇧⌘V`    |          마크다운 미리보기 열기           |
|   `⌘K V`    |    마크다운 미리보기 화면 분할로 열기     |
|    `⌘J`     |             하단 영역 여/닫기             |
|     ⌃`      |             통합 터미널 보기              |
|     ⌃⇧`     |              새 터미널 생성               |

## 에디터 내 편집

|       단축키       |               설명                |
| :----------------: | :-------------------------------: |
|        `⌘X`        |           라인 잘라내기           |
|        `⌘C`        |             라인 복사             |
|     `⌥↓ / ⌥↑`      |        라인 아래, 위 이동         |
|    `⇧⌥↓ / ⇧⌥↑`     |        라인 아래, 위 복사         |
|       `⇧⌘K`        |             라인 삭제             |
| `⌘Enter / ⇧⌘Enter` |        라인 삽입 위, 아래         |
|     `⌘] / ⌘[`      |        들여쓰기, 내어쓰기         |
|     `⌘↑ / ⌘↓`      |     파일의 시작, 끝으로 이동      |
|    `⌥⌘[ / ⌥⌘]`     |      접이식 영역 접기/펼치기      |
|  `⌘K ⌘[ / ⌘K ⌘]`   | 모든 하위 접이식 영역 접기/펼치기 |
|  `⌘K ⌘0 / ⌘K ⌘J`   |       모든 영역 접기/펼치기       |
|        `⌘/`        |          라인 설명 토글           |
|       `⇧⌥A`        |          블럭 설명 토글           |

## 에디터 화면 관리

|      단축키       |                      설명                      |
| :---------------: | :--------------------------------------------: |
|       `⌘\`        |          현재 파일을 가지고 화면 분할          |
|  `⌘1 / ⌘2 / ⌘3`   |        1/2/3번째 분할 화면 그룹 포커싱         |
| `⌘K ⇧⌘← / ⌘K ⇧⌘→` |         분할 화면 내 순서 앞/뒤로 이동         |
|  `⌘K ⌘← / ⌘K ⌘→`  |        이전/다음 분할 화면 그룹 포커싱         |
|   `⌘K ← / ⌘K →`   |       포커싱된 분할 화면 그룹 위치 이동        |
|  `⌘(⇧)+ / ⌘(⇧)-`  |                   확대/축소                    |
|    `⌃⌘← / ⌃⌘→`    | 포커싱된 화면 왼쪽/오른쪽의 분할 화면으로 이동 |

## 파일 관리

|    단축키    |                설명                 |
| :----------: | :---------------------------------: |
|     `⌘N`     |               새 파일               |
|     `⌘O`     |              파일 열기              |
|     `⌘S`     |                저장                 |
|    `⇧⌘S`     |           다른 이름 저장            |
|    `⌥⌘S`     |              전체 저장              |
|     `⌘W`     |                닫기                 |
|   `⌘K ⌘W`    |      열려 있는 파일 전체 닫기       |
| `⌃™ / ⌃⇧Tab` |          다음/이전 탭 열기          |
|    `⌘K P`    |   포커싱하고 있는 파일 경로 복사    |
|    `⌘K R`    | 탐색기로 포커싱하고 있는 파일 표시  |
|    `⌘K O`    | 포커싱하고 있는 파일 새 창에서 보기 |

## 선택, 다중 선택

|      단축키       |                     설명                      |
| :---------------: | :-------------------------------------------: |
|    `⌃⇧⌘→ / ←`     |                선택 확대, 축소                |
|    `⌥ + click`    |              클릭으로 다중 선택               |
| `⇧⌥ + drag mouse` |              드래그로 다중 선택               |
|    `⌥⌘↑ / ⌥⌘↓`    | 선택된 범위 기준 위/아래로 늘려가며 다중 선택 |
|   `⇧⌥⌘↑ / ⇧⌥⌘↓`   |          커서 기준 위/아래 다중 선택          |
|       `⌘U`        |             마지막 커서 실행 취소             |
|       `⇧⌥I`       |          선택한 각 줄 끝에 커서 삽입          |
|       `⌘D`        |                같은 단어 선택                 |

## 제안 및 힌트

|  단축키   |            설명             |
| :-------: | :-------------------------: |
| `⌃Space`  |         트리거 제안         |
| `⇧⌘Space` |    트리거 파라미터 힌트     |
|   `⇧⌥F`   |          문서 포맷          |
|  `⌘K ⌘F`  |          선택 포맷          |
|   `F12`   |     정의한 위치로 이동      |
|  `⌥F12`   |    정의한 내용 바로보기     |
| `⌘K F12`  | 정의한 내용 사이드에서 열기 |
|  `⇧F12`   |        레퍼런스 보기        |
|   `⌘`.    |          빠른 수정          |


## 자동 완성 및 변경

| 단축키  |           설명            |
| :-----: | :-----------------------: |
|  `F2`   |      변수 이름 변경       |
| `⌘K ⌘X` |     뒤 쪽의 공백 제거     |
|  `⇧⌥O`  | 사용되지 않는 import 제거 |

## 안내

|   단축키   |                  설명                  |
| :--------: | :------------------------------------: |
|    `⌃G`    |           특정 라인으로 이동           |
|   `⇧⌘O`    |         특정 변수 위치로 이동          |
|   `⇧⌘M`    |           Problems 패널 보기           |
| `F8 / ⇧F8` |      다음/이전 에러, 경고로 이동       |
|  `⌃⇧Tab`   | 화면 분할 시 분할된 그룹 히스토리 탐색 |
| `⌃- / ⌃⇧-` |         이전/앞 포커스로 이동          |
